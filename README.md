## 실행 방법

#### 1. 레포지토리 클론

```bash
git clone https://github.com/jelly12paw/UpNote_CloneCoding.git
```

#### 2. 폴더 이동

```bash
    cd upnote
```

#### 3. 패키지 설치

```bash
    yarn install
```

##### npm 설치 시 **.yarn**, **yarn.lock**, **.yarnrc.yml** 폴더 삭제 후 진행
```bash
    npm install
```

#### 4. 실행하기

```bash
    yarn start
```

```bash
    npm start
```
